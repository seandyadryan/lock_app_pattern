Kebijakan Privasi

Lock App Pattern
Package name: com.deploydulupulangnanti.lockapp
Berlaku sejak: 30 Juni 2026

Ringkasan

Lock App Pattern adalah aplikasi pengunci dengan pola. Aplikasi ini dirancang untuk membantu pengguna melindungi akses ke aplikasi atau area tertentu di perangkat.

Versi aplikasi saat ini tidak mengumpulkan, mengirim, menjual, atau membagikan data pribadi pengguna kepada pengembang maupun pihak ketiga.

Data yang Dikumpulkan

Aplikasi ini tidak mengumpulkan data pribadi seperti nama, email, nomor telepon, lokasi, kontak, foto, pesan, atau informasi identitas lainnya.

Pola Kunci

Pola kunci yang dibuat pengguna digunakan hanya untuk membuka aplikasi. Pola tersebut diproses di perangkat pengguna dan tidak dikirim ke server pengembang.

Pada versi saat ini, pola aktif selama sesi aplikasi berjalan. Jika pada versi berikutnya aplikasi menambahkan penyimpanan lokal permanen, data pola tetap akan disimpan di perangkat pengguna dan tidak dikirim ke server pengembang.

Izin Perangkat

Versi aplikasi saat ini tidak membutuhkan izin khusus untuk mengumpulkan data pengguna.

Jika di masa depan aplikasi menambahkan fitur pengunci aplikasi lain secara penuh di Android, aplikasi mungkin meminta izin tambahan seperti akses penggunaan aplikasi, layanan aksesibilitas, atau izin tampil di atas aplikasi lain. Izin tersebut hanya akan digunakan untuk menjalankan fungsi penguncian aplikasi di perangkat pengguna.

Internet, Analytics, dan Iklan

Aplikasi ini tidak menggunakan koneksi internet untuk mengirim data pengguna.

Aplikasi ini tidak menggunakan layanan analytics, tracking, iklan, atau SDK pihak ketiga untuk memantau aktivitas pengguna.

Berbagi Data dengan Pihak Ketiga

Pengembang tidak menjual, menyewakan, menukar, atau membagikan data pribadi pengguna kepada pihak ketiga.

Keamanan

Kami berupaya menjaga aplikasi agar aman digunakan. Namun, tidak ada metode penyimpanan atau pemrosesan data yang sepenuhnya bebas risiko. Pengguna disarankan untuk menjaga keamanan perangkat, tidak membagikan pola kunci kepada orang lain, dan mengunduh aplikasi hanya dari sumber resmi.

Privasi Anak

Aplikasi ini tidak ditujukan untuk mengumpulkan data dari anak-anak. Karena aplikasi tidak mengumpulkan data pribadi, pengembang tidak secara sengaja mengumpulkan informasi pribadi dari anak-anak.

Perubahan Kebijakan Privasi

Kebijakan privasi ini dapat diperbarui jika fitur aplikasi berubah. Perubahan akan ditampilkan melalui pembaruan dokumen ini.

Kontak

Jika ada pertanyaan tentang kebijakan privasi ini, silakan hubungi pengembang melalui halaman GitHub repository aplikasi ini.
